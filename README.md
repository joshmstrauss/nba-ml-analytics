# nba-ml-analytics

This NBA Analytics app/platform is meant to be a fun personal project. The goal is to be able to display all sorts of NBA stats, and eventually to implement ML models to predict regular season awards such as MVP, All-NBA, DPOY, etc. Thanks for reading, and Thunder in 4. 


General Data Pipeline:  (this pipeline is a work in progress)
1. Fetch data from nba_api
2. Validate raw data
3. Store raw data to csv
4. Clean and transform data into processed tables
5. Create feature tables for analytics and models
6. Serve/display those features in app visualizations
